# 命宫卡牌设计（第二版·资源循环版）

---

## 前言：设计理念重构

### 一、原版问题诊断

| 问题类型 | 具体表现 | 设计影响 |
|---------|---------|---------|
| **资源定位矛盾** | 几乎所有卡牌都"消耗气运"获得效果 | 玩家无法保留高气运，战后奖励机制形同虚设 |
| **获取方式不足** | 气运获取主要靠功法被动，主动获取方式少 | 气运循环断裂，形成"只消耗不补充"的负循环 |
| **次级资源割裂** | 道具/控制/霉运与气运是单向消耗关系 | 次级资源无法反哺核心资源，博弈深度不足 |
| **宝物/控制品类过多** | 宝物卡和控制卡挤占交互空间 | 变成机械的生成-使用，缺乏策略乐趣 |

### 二、核心改进思路

**核心理念**：将"获取次级资源"与"消耗气运"分离

| 操作类型 | 气运变化 | 效果强度 |
|---------|---------|---------|
| **基础效果** | 获得气运或无消耗 | 标准强度 |
| **强化效果** | 消耗气运 | 超模强度 |
| **终极效果** | 大量消耗气运 | 翻盘效果 |

**品类简化**：
- **多宝系列**：每个品级1张宝物卡，由玩家选择宝物类型
- **趋吉避凶系列**：只保留眩晕、沉默两种控制效果

**资源循环设计**：

```
┌─────────────────────────────────────────────────────────────────────┐
│                     命宫气运循环系统                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────┐                                                   │
│   │    气运     │ ←── 施加控制 ←── 施加霉运 ←── 使用道具            │
│   │  (0-100)   │         ↑              ↑              ↑           │
│   └─────────────┘         │              │              │           │
│         │                 │              │              │           │
│         ↓                 │              │              │           │
│   ┌─────────────┐         │              │              │           │
│   │  消耗气运   │ ──→ 延长控制 ──→ 引爆霉运 ──→ 强化道具            │
│   └─────────────┘                                                   │
│         │                                                           │
│         ↓                                                           │
│   ┌─────────────┐                                                   │
│   │  战后奖励   │ ←── 保留气运获得额外奖励                           │
│   │ (21-100档) │                                                    │
│   └─────────────┘                                                   │
│                                                                     │
│   策略核心：                                                        │
│   · 战斗轻松路线：消耗气运强化效果，牺牲战后奖励                     │

│   · 平衡路线：适当消耗，确保胜利同时保留一定气运                     │
```

### 三、与宗门设定的契合

**命宫价值观**："天机渺渺，逆天改命"

| 学派 | 气运获取方式 | 气运消耗方式 | 主题诠释 |
|------|-------------|-------------|---------|
| **多宝** | 使用道具获得气运 | 消耗气运强化/复制道具 | "以物养运，以运换物" |
| **趋吉避凶** | 施加控制获得气运 | 消耗气运延长/传播控制 | "以控养运，以运控场" |
| **咒杀** | 施加/触发霉运获得气运 | 消耗气运引爆/传播霉运 | "以咒养运，以运咒杀" |

### 一、核心机制说明

#### 1. 气运系统

- **定义**：命宫的"天命之力"，代表窥探天机所获的气运
- **初始值**：0，上限100
- **基础效果**：战斗结束后根据剩余气运获得额外奖励
  | 剩余气运 | 奖励加成 |
  |---------|---------|
  | 0-20 | 无额外奖励 |
  | 21-40 | 额外获得1次普通奖励 |
  | 41-60 | 额外获得1次稀有奖励 |
  | 61-80 | 额外获得1次史诗奖励 |
  | 81-100 | 额外获得1次传说奖励 |
- **衰减机制**：每回合结束衰减5%（造成伤害则衰减2.5%）
- **战斗结束**：清空

#### 2. 控制效果说明

**命宫只保留两种核心控制效果**：

| 控制类型 | 效果 | 战术价值 |
|---------|------|---------|
| **眩晕** | 无法行动1回合 | 打断敌人节奏 |
| **沉默** | 无法使用法术2回合 | 封印敌人技能 |

**其他debuff（易伤、破甲、虚弱等）由法术牌的Buff/Debuff类卡牌提供。**

#### 3. 道具类型说明

**道具分为三类，玩家使用宝物卡时自主选择**：

| 道具类型 | 代表道具 | 效果方向 |
|---------|---------|---------|
| **丹药** | 疗伤丹、回气丹 | 回复生命/灵气 |
| **符箓** | 爆裂符、定身符 | 造成伤害/控制 |
| **防具** | 金刚符、护心镜 | 护甲/免伤 |

### 六、设计改进说明

#### 1. 多宝系列改进

| 改进项 | 原版 | 第二版 |
|--------|------|--------|
| **宝物获取卡** | 10张（每种道具一张卡） | 4张（每个品级一张，玩家选择道具类型） |
| **道具交互卡** | 8张 | 14张（增加道具精通、丹药/符箓/防具精通、道具连发等） |
| **策略深度** | 机械生成-使用 | 道具类型选择+道具强化+道具组合 |

**新增道具交互卡牌**：
- 道具精通（使用道具获得气运）
- 丹药/符箓/防具精通（特定类型道具强化+气运）
- 道具连发（本回合多用道具）
- 宝物转化（道具类型转换）
- 道具爆发（一次性使用所有道具）

#### 2. 趋吉避凶系列改进

| 改进项 | 原版 | 第二版 |
|--------|------|--------|
| **控制效果类型** | 8种（眩晕、沉默、缴械、易伤、禁疗、恐惧、破甲、定身） | 2种（眩晕、沉默） |
| **控制获取卡** | 10张（每种控制一张卡） | 4张（核心控制卡） |
| **控制交互卡** | 8张 | 14张（增加卦象转化、卦象传播、避凶系列等） |
| **策略深度** | 机械施加控制 | 控制时机+控制延长+控制转化+控制后伤害 |

**新增控制交互卡牌**：
- 避凶系列（控制后伤害加成、控制延长）
- 卦象转化（眩晕↔沉默转化）
- 卦象传播（单体控制→群体控制）

#### 3. 与其他debuff的分离

**控制效果（趋吉避凶专属）**：
- 眩晕、沉默

**通用debuff（法术牌提供）**：
- 易伤、破甲、虚弱、恐惧、碎防、标记

这样设计的好处：
1. 降低认知负担：玩家只需记住两种控制效果
2. 增加策略深度：控制交互卡牌更多，玩法更丰富
3. 保持扩展性：debuff由法术牌提供，不影响核心机制

### 七、与命宫主题的契合

| 学派 | 气运获取体现 | 气运消耗体现 | 主题诠释 |
|------|-------------|-------------|---------|
| **多宝** | 使用道具养运 | 消耗气运强化道具 | "以物养运，以运换物" |
| **趋吉避凶** | 施加控制养运 | 消耗气运延长控制 | "以控养运，以运控场" |
| **咒杀** | 施加霉运养运 | 消耗气运引爆霉运 | "以咒养运，以运咒杀" |

**核心博弈**：
- 消耗气运 → 战斗轻松，但战后奖励差
- 保留气运 → 战后奖励好，但战斗困难
- 玩家需要根据当前状态（血量、卡组成型度、后续敌人强度）做出选择
